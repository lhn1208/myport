맞춤단체 단독여행(프라이빗) grouptour.html grouptour.html
  • 문서타입 : HTML5
  • 언어 속성 : ko
  • 문자열 인코딩 : UTF-8
  • 브라우저 지원범위 :

    ie >= 10,
    last 10 Chrome versions,
    last 10 Firefox versions,
    last 2 Opera versions,
    iOS >= 6, Android >= 4.3

    기본적으로 Can I Use(http://caniuse.com/)에 근거해서 작업하고, CSS Vender prefix는 브라우저 지원범위 설정에 따라 gulp-autoprefixer에서 자동으로 처리되므로 소스파일에서는 표준 속성만 사용한다.

  • 사용 라이브러리 : jquery 2.1.0
  • 사용 플러그인 : 스와이퍼 http://www.idangero.us/swiper/#.VpM8H7aLQwE
  • 코딩환경 :

    프로젝트 소스 빌드를 위해 node.js(node-v10.15.3)와 gulp를 사용합니다.
    코딩 환경을 실행하고 소스파일을 빌드해서 배포용 HTML, CSS를 생성하는 구조입니다.

    프로젝트에서 사용되는 주요 모듈은 아래와 같습니다

    모듈명설명
    gulptask 자동화 도구
    gulp-html-tag-includehtml 모듈화를 위한 템플릿 엔진
    gulp.spritesmith이미지 스프라이트 및 스프라이트 CSS 자동 생성
    gulp-sassCSS 전처리기 SASS 사용
    gulp-autoprefixerCSS prefix 자동화
    browser-sync로컬 웹서버

    gulp 참고 사이트 : http://programmingsummaries.tistory.com/356

    scss 참고 사이트 : https://sass-lang.com/guide

  • 코딩환경설치:

    node.js
    node.js 웹사이트(https://nodejs.org/)에서 다운로드 후 설치

    gulp
    명령프롬프트에서 gulp 실행을 위해 global 옵션으로 설치합니다.

    npm install -global gulp

    Install
    package.json이 있는 프로젝트 폴더에서 명령프롬프트를 열고 아래 명령을 실행하면 프로젝트에 필요한 모듈이 설치됩니다.

    npm install

  • 코딩환경 실행 :

    gulpfile.js가 있는 프로젝트 폴더에서 명령프롬프트를 열고 gulp 명령어를 실행합니다.

    실행하고자 하는 프로젝트 이름을 gulpfile.js에서 var myRoot = ""; 괄호안에 프로젝트 이름으로 실행시켜야 합니다. ex)예약페이지 var myRoot = "reserve";

    gulp

    기본으로 default task가 실행되면서 watch 상태가 되어 소스파일이 변경되면 자동으로 컴파일됩니다. 프로젝트에 구성된 Task 정보는 아래와 같습니다.

  • 스타일 가이드

    이미지 파일은 적절한 영역별 폴더구분하고, 파일명 단어 구분은 언더스코어(_)로 한다.
    고해상도 이미지는 2배수 이미지로 취급하여 1/2 사이즈로 적용한다.
    스프라이트 이미지는 자동생성되어 관리된다.
    CSS 클래스는 소문자로만 작성하며 단어 구분은 하이픈(_)으로 한다.
    CSS 셀렉터 중첩이 4단계 이상 깊어지지 않도록 주의한다.
    SCSS 작성 시 CSS 벤더프리픽스는 자동으로 삽입되므로 프리픽스 사용은 하지 않는다.
    믹스인과 함수의 사용 효율보다 읽기 쉬운 코드와 일관성이 우선하므로 과도한 믹스인은 자제한다.
