더샵플러스 엔터테인먼트 서비스 18년 1월 오픈!

무료 영화부터 최신 신작 영화까지, 원하는 간식과 음료까지 다양한 서비스를 더샵플러스 포인트로 이용 하실 수 있습니다. 엔터테인먼트 퀴즈 참여를 통해 100% 적립 + 추첨을 통해 더샵플러스 포인트를 적립해서 엔터테인먼트 서비스를 이용해 보세요! 지금 놓치지 마시고 바로 참여하세요!

이벤트 참여방법
  1. 퀴즈 이벤트에 참여한다.
  2. 퀴즈 참여 시 100% 더샵플러스 포인트를 지급받는다.
  3. 퀴즈 참여 시 엔터테인먼트 관련 의견을 남기고 추첨을 통해 추가 지급받는다.
  4. 지급받은 포인트로 영화시청, 기프티콘으로 교환하여 사용한다.
엔터테인먼트 서비스란?
영화 서비스
무료 영화 250편 + 성인영화50편에 최신 영화(더샵플러스포인트 이용가능)까지 이용가능한 코너 입니다.
기프티콘(SK플래닛)
커피, 치킨, 피자, 상품권 등 약 285개의 브랜드 상품을 더샵플러스 포인트로 구매할 수 있는 모바일 상품권 코너입니다.
더샵플러스 엔터테인먼트 서비스 이용방법 더샵플러스 (좌측메뉴) 접속 방법
  1. 더샵 메인 접속
  2. 좌측 메뉴 중 더샵 플러스로고 클릭
  3. 클릭 시 더샵플러스 서비스 항목 펼쳐짐
영화 시청 방법
  1. 좌측 메뉴 엔터테인먼트 접속 영화 메뉴 접속
  2. 더샵전용관 에서 원하는 영화 선택 후 무료보기 클릭 시 바로 시청 가능(단, 다운로드 클릭 시 포인트 차감)
  3. 최신 영화는 원하는 영화 선택 후 바로보기 클릭 시 결제진행 팝업 노출
  4. 더샵플러스 포인트로 결제진행 완료 후 바로보기를 통해 바로 시청 가능(포인트 부족 시 시청 불가)
기프티콘 교환 방법
  1. 좌측 메뉴 '엔터테인먼트' 접속 '기프티콘' 메뉴 접속
  2. 원하는 상품 선택 후 선물하기 또는 구매하기 클릭 시 구매 및 발송 페이지로 이동
  3. 받는 분 정보 입력 후 다음단계 클릭하여 결제 페이지로 이동
  4. 결제방식 더샵플러스 포인트 선택 결제하기 클릭 시 노출되는 팝업에서 포인트 적용하여 결제
이벤트1 참여만해도 더샵플러스 포인트 2000점 100% 적립 이벤트 더샵플러스 포인트로 적립되어집니다.
엔터테인먼트 퀴즈 이벤트 입력폼 퀴즈1 더샵플러스 포인트로 이용이 가능한 서비스를 선택해 주세요. 중복 선택
