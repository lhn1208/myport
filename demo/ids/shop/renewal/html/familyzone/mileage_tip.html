특별한 혜택공간 마일리지 사용TIP

TheSHOP 회원께 제공하는 다양한 마일리지 서비스를 확인해보세요.

  • 모바일 상품권 교환

    모바일 상품권 교환하기
    1. 원하는 카테고리를 선택
    2. 상품권 종류와 수량을 선택 후 모바일상품권으로 교환
    3. 신청하신 모바일상품권은 신청일 기준으로 차주에 발송 예정

    * 매월 3주차, 4주차 신청 건은 익월 첫째 주 수요일에 발송 예정이니다.

  • 예치금 상품권 교환

    예치금 상품권 교환하기
    1. 더샵 마일리지를 예치금 상품권으로 교환
    2. 최소 1만원 단위로 교환 가능
      ex) 10,000원권 X ___ 매
    3. 교환된 예치금 상품권은 더샵 마일리지로 재교환 불가능
    4. 예치금 상품권 유효기간은 없으며, 예치금 환급 불가

  • 부외품 쿠폰 전환

    부외품 쿠폰 전환하기
    1. 더샵 마일리지를 부외품 쿠폰으로 전환
    2. 최소 1천원 단위로 전환 가능
      ex) 1,000원 X ___ 매 / 5,000원 X ___ 매
    3. 쿠폰 적용 금액 제한은 없으며, 유효기간은 전환일로부터 3개월
    4. 전한된 쿠폰은 더샵 마일리지로 재전환 불가능

  • 낱알반품 포인트 전환

    포인트전환하기 낱알반품 바로가기
    1. 더샵 마일리지 5,000점을 낱알반품 포인트 5,000포인트로 전환
    2. 마이페이지 > 나의 반품내역 >
      도매/부외품반품 > 낱알반품신청       메뉴에서 전환된 포인트만큼 자유롭게 반품신청 가능
    3. 더샵에서 구매이력이 없는 제품도 반품 가능
      단, 서울지오팜, 인천약품 주문이력 최소 1회 이상 있는 경우에만 반품 가능
