  • 신청방법
    1. 반품추가 버튼 클릭 > 팝업에서 상품 선택 > 선택추가 버튼 클릭 > 닫기 버튼 클릭
    2. 반품사유, 반품 박스수, 반품하실 수량을 선택 또는 입력하신 후 하단의 반품신청을 누르시면 반품접수가 완료됩니다.
    3. 향정의약품 반품신청시, 보건소의 승인을 받은 후 반품이 진행되어야 하오니, 아래 절차와 필요서유를 확인하신 후, 반품 신청을 해주시기 바랍니다.
      - 향정의약품 (디에타민정,졸피아트정,지미콜정,베아콜정,베아콜에프정,베아코프에프정)
      • 1) 향정의약품 반품신청시 영업사원에게 연락하여 (SMS발송), 아래 서류 2가지 원본을 받아야 합니다.
        • ① 마약류 양도승인 신청서 출력
        • ② 계약서 작성 및 출력 후 대웅제약 사용인감 날인 (원본)
          * 날인 전 작성내용 : 양도인란 기재
          * 사용인감 날인 : 영업사원이 해당관리과에 요청하여 처리후 약국에 전달
      • 2) 해당 시군구 보건소에 승인신청 및 승인서류 접수
      • 3) 반품신청
        ※ 동봉서류 : ① 환입명세서 ② 계약서 사본 ③ 보건소승인서류 사본
      • 4) 반품회수
      • 5) 환입실 실물 &서류 확인 후 반품전산처리
        * 서류 불충분시 반송처리 / 서류 충분시 마이너스잔고로 반품정상처리
    영업사원에게 SMS무료발송
    참고사항
    1. 반품 신청시 배송사원이 방문하여 3일이내 회수처리합니다.
    2. 반품 회수 후 최대 11일안에 마이너스 잔고로 반영되며 최종 금액은 마이페이지 내 거래원장에서 확인 가능합니다.
      (단, 월말에는 처리 시간이 더 걸릴수 있습니다.)
    3. 보여드리는 예상 반품 금액에는 입력하신 낱알 반품 예상 금액도 반영되어 확인됩니다.
      하지만 특정 상품일 경우 낱알 갯수 산정이 안될수 있으니 이점 참고 부탁드립니다.
    4. 현재 판매되고 있지 않은 상품인 경우. 반품예상금액이 '-' 로 표시될 수 있습니다.
    5. 반품신청 내역과 실물이 일치하도록 해주세요.
      반품신청후 수량/제품 변동시에는 제품 인수인계 하시기전 환입명세표에 수정 서명 바랍니다.
    6. 낱알수량 입력방법 : 예) 우루사 10캡슐* 12개 -> 낱알수량 120개로 입력
    반품신청시 필독

    * 대포장제품으로 구입 후 판매하시다가, 낱개제품에 대하여 반품신청시 '주문한 내역이 없는 제품'으로 인식되오니,
    대포장제품으로 구입하신 제품은 반드시 대포장약호로 반품요청 하시기 바랍니다.

    ex) 120정(30개) 대포장 제품으로 1셋트 주문 후, 120정 완포장 1개 반품 신청시 20정 제품으로 완제품수량 1개로 입력하면 안되며, 120정(30개) 대포장 제품으로 낱알수량에 120개로 반품수량 입력해 주세요.

    • - 2017년 1월이후부터는 주문이력 없는 반품처리는 불가함
    • - 구매일로부터 직전 3년이내 주문한 수량 만큼만 반품신청이 가능

    2. 반품할 상품 신청수량 입력후 선택상품 반품추가 클릭 (반품신청가능수량만 반품신청가능)
    3. 낱알수량입력또는 반품사유 선택
    4. 최종 반품선택 상품 선택후 반품신청 클릭
