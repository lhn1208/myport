고객이 만족하는장기렌탈 서비스사용후기

CASE 1.A병원 내과 원장님
“신 장비를 비용 부담 없이 도입하여 제공받은 꼼꼼한 운영 가이드로 쉽게 이용하고 있습니다”

신 장비를 매월 렌탈 비용만 내고 부담 없이 이용할 수 있는 것도 좋지만, 가장 좋은 점은 어떤 환자군을 대상으로 어떻게 검사하는지 알려주는 것은 물론 환자들에게 홍보할 수 있는 안내자료와 사용 매뉴얼을 꼼꼼하게 챙겨줘서 쉽게 장비를 운영하고 있습니다.

CASE 2.B병원 내과 원장님
“A/S에 대한 스트레스가 없습니다”

모든 병원이 동감하겠지만 장비 운영하면서 A/S가 항상 골치였습니다. 장비업체가 고장 사항에 대해 빨리 피드백을 주지 않고 병원 과실이 아닌데도 전부 비용을 청구해서 장비 운영에 따른 A/S 비용도 많이 들었습니다.

그런데, 이제 렌탈사의 콜센터를 통해 답변도 빠르게 받을 수 있고 A/S 기간도 책임지고 3년 동안 지원해주는데다 동일한 고장이 3번 나면 다른 장비로 교체해주는 서비스까지 있으니 이제 걱정 없이 장비 운용할 수 있을 것 같습니다.

이용안내

  1. 신규상담

    전화상담을 신청하면
    비용 및 렌탈/구매 방법에
    대해 친절히 안내드립니다.

  2. 계약

    상담 후 렌탈을 원하는 경우
    직원이 병원으로 방문하여
    계약서를 작성합니다.

  3. 배송/설치

    렌탈한 의료기기를
    전문 엔지니어가
    배송 및 설치해드립니다.

  4. 이용

    이용하면서 불편한 점이나
    기기에 문제가 발생한 경우
    A/S 신청이 가능합니다.

  5. 만기/중도 상담

    계약만기 시 소유권이 이전되며 이용 중에도 언제든지 소유권 이전이 가능합니다.
    ※ A/S연장 또한 가능합니다.
