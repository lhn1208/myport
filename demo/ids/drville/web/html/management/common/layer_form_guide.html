문서 사용 안내
  • 워드

    편집제한이 설정되어 있어, 노란색으로 활성화된 부분만 작성이 가능합니다.
    노란색으로 활성화된 부분이 없는 문서는 편집이 되지 않는 출력전용 문서입니다.

  • 엑셀

    시트 보호가 되어 있어, 빈 셀에만 작성이 가능합니다.

  • 한글

    전체 편집이 가능합니다. 만약 편집되지 않는 문서는 출력전용 문서입니다.

  • PDF

    편집이 불가능한 출력전용 문서입니다.
